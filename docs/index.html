Apiko - The easiest way to create a REST API server

OK well. How long does it take to learn?

Wrong question. You can already create projects with Apiko.

All you need to start creating is to do:

git clone https://github.com/kasp1/apiko-start.git
cd apiko-start
npm install
npm run dev

Apiko then opens in a browser tab. If it doesn't, go to http://localhost:5000/dev/.
